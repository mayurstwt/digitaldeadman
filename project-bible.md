# Project Bible: Digital Deadman

## One-sentence mission
"Build a payment-locked security script for freelance developers to ensure they get paid for their work after site handover."

## Target user + core value
"Freelance web developers who get ghosted by clients → security and leverage to ensure final payments."

## Non-goals
"No dashboard yet, no social sharing, no gamification badges, no complex multi-user management in MVP."

## Tech stack rules
"Next.js 15 app router + Tailwind + shadcn/ui + Supabase auth/DB + Stripe. No extra libs unless critical."

## Design rules
"Clean, minimal, dark mode default, mobile-first. Use shadcn components only."

## Naming conventions
"Components: PascalCase, files: kebab-case, env vars: UPPER_CASE"

## Success criteria
"MVP live when: Landing page with headline and buy button is live, validation of interest is achieved via clicks."

## AI Prompt Instructions
"@project-bible.md Read this file first. Follow every rule strictly. Do NOT add features outside the mission."
